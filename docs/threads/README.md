# Bib Thread Bootstrap

以下文件用于在 Codex 中快速新开独立 thread。  
每个文件对应一个 thread 的首条消息模板。

命名规则建议  
`[bib][YYYY-MM-DD][area] short-goal`

推荐 thread 列表

1. `01-ui-layout.md`
2. `02-ui-interaction.md`
3. `03-candidate-flow.md`
4. `04-api-backend.md`
5. `05-tests-regression.md`
6. `06-release.md`

Manager thread 模板

1. `manager-ui-interaction.md`
2. `manager-candidate-flow.md`
3. `manager-api-backend.md`
4. `manager-tests-regression.md`
5. `manager-release.md`

使用方式

1. 新开一个会话 thread
2. 复制对应文件全部内容作为首条消息
3. 在该 thread 内只做一个目标
