# Changelog

## [Unreleased]
### 新增
- 新增 UI 检索进度条和阶段提示文案

### 优化
- 将检索和候选确认流程改为后台线程执行  
  解决检索期间主窗口无响应问题
- 增加检索单路超时控制  
  默认 `search_timeout_sec=6.0`，避免单个数据源超时拖慢整体

### 测试
- 本地通过 `pytest` 回归测试
- 本地通过 UI 模块编译检查

## [0.1.2] - 2026-04-20
### 修复
- 修复 Vancouver 风格参考文献识别，避免将作者段误判为标题输入
- 修复年份在末尾时的解析逻辑，正确提取作者 标题 年份
- 修复部分参考文献在检索不稳定时直接失败的问题  
  现在会提供可确认的无 DOI 兜底候选
- 修复 BibTeX 输出偶发单行问题  
  规范化 `month=june` 等字段后统一输出多行
- 修复 Crossref 请求异常导致的检索中断  
  请求失败时自动回退为空结果并继续流程
- 修复候选自动通过规则过严的问题  
  对明显领先的 DOI 候选支持自动成功

### 测试
- 增加并通过回归测试  
  覆盖参考文献识别、解析、匹配和 BibTeX 格式化

## [0.1.1] - 2026-04-12
### 修复
- 修复 BibTeX key 规则 `作者姓+年份+标题首词` 中标题首词首字母为小写的问题  
  输出统一为 `Zhou2007Functional`

## [0.1.0] - 2026-04-11
### 新增
- 单条输入 MVP
- 支持 DOI、标题、完整参考文献字符串输入
- 支持 Crossref 和 OpenAlex 双源检索
- 支持 BibTeX 生成、复制和 key 规则切换
